# Generated SolidStart HackerNews app

This app is generated from `src/Examples/SolidStart` and targets `@solidjs/start` alpha + Nitro.

- Edit PureScript app logic under `src/Examples/SolidStart`.
- Regenerate app glue with `npm run gen:example:solid-start-app`.
- Install app dependencies with `npm run install:example:solid-start`.
- Run dev server with `npm run dev:example:solid-start`.
- Build production output with `npm run build:example:solid-start`.

Node.js `>=22` is required by the Start alpha runtime.

Do not hand-edit `examples/solid-start/` unless you are changing the generator/template.
